## 📁 Projects Overview

| # | Project | Dataset | Classes | Best Accuracy |
|---|---------|---------|---------|---------------|
| 1 | [Animal Classification](./Task1-Animal-Classification/) | Animals-10 (Kaggle) | Cat, Dog, Horse, Elephant | 76.11% |
| 2 | [Fashion-MNIST Optimization](./Task2-Fashion-MNIST-Optimization/) | Fashion-MNIST (Built-in) | 10 Clothing Items | ~93% |
| 3 | [Object Recognition](./Task3-Object-Recognition/) | Natural Images (Kaggle) | Car, Motorbike, Airplane, Person, Dog | ~90% |

## 📊 Results at a Glance
```
Task 1 — Animal Classification
  Basic CNN          →  71.67%
  CNN + Dropout      →  76.11%  🏆
  CNN + Augmentation →  75.17%

Task 2 — Fashion MNIST
  Baseline CNN       →  ~88%
  Extra Conv Layer   →  ~90%
  CNN + Dropout      →  ~91%
  Batch Normalization→  ~93%   🏆

Task 3 — Object Recognition
  No Augmentation    →  ~75%
  With Augmentation  →  ~90%   🏆
```
